# Web Localizr

## Install WebLocalizr

 * install [node.js](http://nodejs.org/)
 * Run `npm install -g weblocalizr`


## Prepare your application

Add this HTML code to your webpage header:

```
  <script src="https://localhost:30443/javascripts/inject.js"></script>
```
